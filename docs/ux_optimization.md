- 增加**底部状态栏**（总线占用率、警告数量实时显示）

### 优化 2：驱动拓扑切换 — 确认对话框

**触发场景**: 用户点击"驱动拓扑"下拉框并选择不同类型时

```
┌──────────────────────────────────────────────┐
│  ⚠️  切换驱动拓扑                             │
│                                              │
│  您正在将驱动类型从:                           │
│    双舵轮 (2轮已配置)                         │
│  切换为:                                      │
│    四舵轮                                     │
│                                              │
│  ────────────────────────────────────────    │
│  此操作将:                                    │
│  • 重置所有轮组参数至默认值                    │
│  • 无法通过撤销恢复当前轮组配置 ⚠️             │
│                                              │
│  建议先保存当前配置再切换。                    │
│                                              │
│    [保存并切换]   [直接切换]   [取消]          │
└──────────────────────────────────────────────┘
```

**交互细节**:
- "保存并切换": 触发另存为对话框，保存后执行切换
- "直接切换": 立即执行，在 Undo 历史中记录为一条可撤销操作
- "取消": 关闭对话框，不做任何改变

### 优化 3：实时校验内联错误提示

**触发场景**: 用户在 CAN Node ID 字段输入与其他轮组相同的值

```
┌──────────────────────────────────┐
│ CAN 节点 ID                       │
│ ┌────────────────────────────┐   │
│ │ 1                    ❌   │   │ ← 输入框变红框
```

### 优化 5：可交互 2D 安装蓝图

**正常状态**：显示组件图标（同 V3）

**进入编辑模式**（顶栏切换按钮"查看模式 / 编辑模式"）：

```
┌─────────────────────────────────────────────────────────┐
│  2D安装蓝图  [查看模式 ●]  [←→编辑模式]                  │
│  ─────────────────────────────────────────────────────  │
│                                                         │
│              ▲ 前 (Head)                                │
│         ┌────────┐                                      │
│         │        │   ⟳ ← 旋转手柄 (拖动改 Yaw)          │
│  ⚙️[FL]─┼──[FR]⚙️│                                      │
│         │ Custom │ ← 拖动机器人整体重新定位               │
│  ⚙️[RL]─┼──[RR]⚙️│                                      │
│         └────────┘                                      │
│              📡 ← 拖动改变X/Y，旋转手柄改Yaw             │
│                                                         │
│  [选中: Front Left ⚙️]                                  │
│  X: [500  mm] Y: [350  mm] Yaw: [0  °]  [应用]         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**编辑模式交互规则**:
- **拖动组件**: 实时更新 XY 数值，松开鼠标时同步到表单并写入 Undo 历史
- **旋转手柄**: 鼠标按住圆形手柄旋转，实时更新 Yaw
- **选中组件**: 底部显示精确数值编辑框（可直接输入数字）
- **按住 Shift**: 拖动时限制为横向或纵向（对齐辅助）
- **右键组件**: 显示快捷菜单（编辑详情 / 删除 / 复制）
- **Tab 键**: 在多个组件间循环选中
